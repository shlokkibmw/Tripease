TripEase tourism web project, the Heritage Cloud page showcases all major monasteries in Sikkim
The Heritage Cloud module is designed to serve as a cultural discovery hub for tourists exploring Sikkim.
It displays monastery information dynamically with interactive features like search and district-based filtering.
Header Section

Displays the project title: TRIPEASE – HERITAGE CLOUD

Contains:
A menu button or hamburger (☰) on the left to toggle the sidebar.
A search bar on the right for typing monastery names.
A filter icon button beside the search bar that opens a dropdown to select districts.
Sidebar Section


Contains profile info and navigation links:
Home
Profile
My Trips
Transactions
Heritage Cloud (current active page)
help
Settings
Main Container
Displays monastery cards in a responsive grid.
Each card contains:
Image of the monastery
Monastery name
Short description


How to use the website:
--> Download or clone this repository.

--> Ensure all image files are present in the same directory.

--> Deploy on vercel or any hosting platform.

--> Use login or register with google to proceed and later use hamburger (☰) to navigate through the website.
